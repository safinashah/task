# 📝 Simple To-Do List App (Python)

A very simple **command-line To-Do List app** made with Python.  
You can add tasks, view tasks, delete tasks, and exit the program.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Delete tasks by number
- Exit the app anytime
